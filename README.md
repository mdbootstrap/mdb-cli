# Supported terminals

### Mac OS:
- Terminal.app
- iTerm
### Windows:
- ConEmu
- cmd.exe
- Powershell
- Cygwin
### Linux (Ubuntu, openSUSE, Arch Linux, etc):
- gnome-terminal (Terminal GNOME)
- konsole
